## 📌 Overview
This lab simulates enterprise network troubleshooting scenarios using DNS, DHCP, packet analysis, and connectivity diagnostics. The lab emphasizes real-world problem-solving and network analysis skills.

---

## 🎯 Objectives
- Diagnose DNS and DHCP issues
- Capture and analyze network traffic with Wireshark
- Identify network connectivity failures
- Troubleshoot using ping and traceroute
- Apply enterprise network troubleshooting methodology

---

## 🧰 Technologies Used
- Wireshark
- Windows Server 2022
- Windows 10/11 clients
- DNS, DHCP
- PowerShell & CMD
- ICMP / TCP / UDP protocols

---

## ⚙️ Lab Sections

### DNS Troubleshooting
- Verify DNS resolution
- Test domain name lookups
- Identify misconfigured DNS settings
- Resolve internal resolution failures

### DHCP Configuration
- Check IP assignment
- Renew IP addresses
- Validate DHCP scope and lease configuration
- Troubleshoot IP conflicts

### Packet Capture Analysis
- Capture live network traffic
- Filter by protocol (TCP/UDP/ICMP/DNS)
- Analyze packet flow and identify anomalies
- Investigate failed requests

### Ping & Traceroute Diagnostics
- Test connectivity to network devices
- Identify latency and routing issues
- Diagnose path failures and packet loss
- Validate gateway and DNS reachability

### Connectivity Troubleshooting
- Confirm client-server communication
- Analyze network topology and routing
- Resolve connectivity issues affecting services
- Document troubleshooting steps
